# VoicePay System

Sistema basado en microservicios que simula un IVR (voice bot) con gestión de pagos y usuarios utilizando Java y Spring Boot.

## Tecnologías
- Java 17
- Spring Boot
- PostgreSQL
- Docker (opcional)

## Servicios
- **user-service**: Gestión de usuarios y perfiles.
- **payment-service**: Procesamiento de pagos e integración con simuladores de pasarela.
- **ivr-service**: Manejo de llamadas entrantes y procesamiento de lógica de voz.
